API Documentation

POST /api/token/
Obtain an access and refresh token.
Request Body:
{
    "username": "string",
    "password": "string"
}
Response:
{
    "access": "string",
    "refresh": "string"
}
POST /api/token/refresh/
Refresh the access token using a refresh token.
Request Body:
{
    "refresh": "string"
}
Response:
{
    "access": "string"
}
POST /api/createUser/

Request Body:

    "username": "string",
    "email": "string",
    "password": "string"
}
Response:

    "email": "string"
}

    "email": "string",
    "is_active": "bool"
